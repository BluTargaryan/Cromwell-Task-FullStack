
# Installation

## Clone the Repository

```bash
git clone <repository-url>

## Open a terminal and navigate to the backend directory
cd backend

## Open the backend directory in your code editor (VS Code)
code .

## Install backend dependencies
npm install

## Run backend server
npm run dev



## Open a terminal and navigate to the frontend directory
cd frontend

## Open the frontend directory in your code editor (VS Code)
code .

## Install frontend dependencies
npm install

## Run frontend server
npm run dev
